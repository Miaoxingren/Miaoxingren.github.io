IE6的双边距BUG

当满足下面这三个条件时，就会出现这个BUG：

  1. 要为块状元素
  2. 要左侧浮动
  3. 要有左外边距margin-left
    解决这个BUG，只需在相应的块状元素的CSS属性中加入display:inline。

JSONP

该协议的一个要点就是允许用户传递一个callback参数给服务端，然后服务端返回数据时会将这个callback参数作为函数名来包裹住JSON数据，这样客户端就可以随意定制自己的函数来自动处理返回数据了。

AJAX与JSONP

  1. AJAX和JSONP这两种技术在调用方式上“看起来”很像，目的也一样，都是请求一个url，然后把服务器返回的数据进行处理，因此jquery和ext等框架都把JSONP作为AJAX的一种形式进行了封装；
  2. 但AJAX和JSONP其实本质上是不同的东西。AJAX的核心是通过XmlHttpRequest获取非本页内容，而JSONP的核心则是动态添加<script>标签来调用服务器提供的js脚本。
  3. 所以说，其实AJAX与JSONP的区别不在于是否跨域，AJAX通过服务端代理一样可以实现跨域，JSONP本身也不排斥同域的数据的获取。
  4. 还有就是，JSONP是一种方式或者说非强制性协议，如同AJAX一样，它也不一定非要用JSON格式来传递数据，如果你愿意，字符串都行，只不过这样不利于用JSONP提供公开服务。

元素嵌套

  1. 块元素可以包含内联元素或某些块元素
  2. 内联元素却不能包含块元素，它只能包含其他的内联元素
  3. DTD中规定了块级元素是不能放在`

    里面的

<html>是在最外层，<html>下一级里面只会有<head>、<body>、<frameset>、<noframes>，而可视的元素只会出现在<body>里，所以把<body>划在第一个级里面，接着，把不可以自由嵌套的元素划在第三个级，其他的就归进第二个级。
所谓的不可自由嵌套的元素就是里面只能放内联元素的，它们包括有：
标题标记的<h1>、<h2>、<h3>、<h4>、<h5>、<h6>、<caption>；
段落标记的<p>；
分隔线<hr>和一个特别的元素<dt>(它只存在于列表元素<dl>的子一级)。
第二级的元素可以自由嵌套，可以把它们看成是一些容器（或者说是盒子）， 这些容器的大小可以自由变化，例如可以把<ul>嵌在<div>里面，也可以把<div>嵌在<li>里面。
在HTML里有几个元素是比较特别的：
<ul>、<ol>、<dl>、<table>，它们的子一层必须是指定元素，
<ul>、<ol>的子一级必须是<li>；
<dl>的子一级必须是<dt>或者<dd>；
<table>的子一层必须是<caption> 或<thead>、<tfoot>、<tbody>等，而再子一层必须是<tr> （<tr>只存在于<thead>、<tfoot>、<tbody>中），之后才是可放内容的<td>或者<th>。

HTML5 Drag 实现

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Drag</title>
    <style type="text/css">
        #drag {
            width: 50px;
            height: 50px;
            background-color: #EEE;
        }
    </style>
</head>
<body>
    <div id="drag"></div>
    <script type="text/javascript">
        window.onload = function () {
            function Drag(obj) {
                this.obj = obj;
            }
            Drag.prototype = {
                constructor: Drag,
                getMosuePos: function(e) {
                    e = e || window.event;
                    var eX = e.pageX || e.clientX + document.body.scrollLeft - document.body.clientLeft,
                        eY = e.pageY || e.clientY + document.body.scrollTop - document.body.clientTop;
                    return {
                        x: eX,
                        y: eY
                    }
                },
                getInitPos: function (e) {
                    e = e || window.event;
                    var eX = e.pageX || e.clientX + document.body.scrollLeft - document.body.clientLeft,
                        eY = e.pageY || e.clientY + document.body.scrollTop - document.body.clientTop;
                    return {
                        x: eX - this.obj.offsetLeft,
                        y: eY - this.obj.offsetTop
                    }
                },
                initDrag: function () {
                    var that = this;
                    this.obj.onmousedown = function (e) {
                        var curPos = that.getInitPos(e);
                        document.onmousemove = function (e) {
                            var mousePos = that.getMosuePos(e);
                            that.obj.style.marginLeft = mousePos.x - curPos.x + "px";
                            that.obj.style.marginTop = mousePos.y - curPos.y + "px";
                        };
                        document.onmouseup = function (e) {
                            document.onmousemove = null;
                            document.onmouseup = null;
                        }
                    };

                }
            };
            var drag = document.getElementById("drag"),
                dragable = new Drag(drag);
            dragable.initDrag();
        };
    </script>
</body>
</html>

XMLHttpRequest对象

  1. 老版本的Internet Explorer(IE5和IE6)使用ActiveXObject。
  2. 所有现代浏览器（IE7+、Firefox、Chrome、Safari以及Opera）均内建XMLHttpRequest对象。

兼容IE：

var xmlhttp;
if (window.XMLHttpRequest) {
    // code for IE7+, Firefox, Chrome, Opera, Safari
    xmlhttp = new XMLHttpRequest();
} else {
    // code for IE6, IE5
    xmlhttp = new ActiveXObject("Microsoft.XMLHTTP");
}
xmlhttp.onreadystatechange = function () {
    if (xmlhttp.readyState == 4 && xmlhttp.status == 200) {
        document.getElementById("myDiv").innerHTML = xmlhttp.responseText;
    }
}

循环中变量赋值问题

变量key随循环结束最终变为z：

function A(params) {
    params = params || {};
    for (var key in params) {
        Object.defineProperty(this, key, {
            get: function() {
                return params[key];
            },
            enumerable: false
        });
    } 
}
var a = new A({
    'x': 'x',
    'y': 'y',
    'z': 'z'
});
console.log(a.x); //'z'
console.log(a.y); //'z'
console.log(a.z); //'z'

使用函数保存变量的值：

function A(params) {
    params = params || {};
    function generateVal(obj, key) {
        Object.defineProperty(obj, key, {
            get: function() {
                return params[key];
            },
            enumerable: false
        });
    }
    for (var key in params) {
        generateVal(this, key);
    } 
}
var a = new A({
    'x': 'x',
    'y': 'y',
    'z': 'z'
});

嵌套数组拆分

function flatten(array) {
    var i,
        res,
        length;
    res = (array + '').split(',');
    for (i = 0, length = res.length; i < length; i++) {
        res[i] = res[i] - "0";
    }
    return res;
}
flatten([1, [2, 3]]); //[1, 2, 3]

function setIframeHeight(iframe) {
    if (iframe) {
        var iframeWin = iframe.contentWindow || iframe.contentDocument.parentWindow;
        if (iframeWin.document.body) {
            iframe.height = iframeWin.document.documentElement.scrollHeight || iframeWin.document.body.scrollHeight;
        }
    }
};

格式化输出

function formater(str) {
    var step = 3,
        tmp = str + '',
        length = tmp.length,
        res = [];
    while (length > 0) {
        if (length < 3) {
            step = length;
        }
        res.unshift(tmp.substring(length - step, length));
        tmp = tmp.substring(0, length - step);
        length = tmp.length;
    }
    return res.join(",");
}

formater(9999999); // "9,999,999"

获取查询参数

function URL(url) {
    this.url = url;
}
URL.prototype.parse = function () {
    var i,
        length,
        pair,
        queries = {},
        queryArray = this.url.split('?')[1].split('&');
    function addParam(key, val) {
        queries[key] = val;
    }
    for (i = 0, length = queryArray.length; i < length; ++i) {
        pair = queryArray[i].split('=');
        addParam(pair[0], pair[1]);
    }
    return queries;
}
var url = new URL( "http://witmax.cn/index.php?key0=0&key1=1&key2=2")
url.parse(); //{key0: "0", key1: "1", key2: "2"}

addEventListener和attachEvent的this指向不同

  1. 标准浏览器中可以使用addEventListener函数来给DOM元素绑定事件，而IE6、IE7、IE8不支持addEventListener，只能使用attachEvent方法。
  2. 标准浏览器中可以使用removeEventListener函数来给DOM元素移除绑定的事件，而IE6、IE7、IE8不支持removeEventListener，只能使用detachEvent方法。
  3. IE6、IE7、IE8使用attachEvent以后事件绑定程序中的this指向出现错误，this指向window。通过使用call或apply可以改变this指针的指向。
